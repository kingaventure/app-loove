# 💘 App-Loove – Application de rencontre

Bienvenue sur **App-Loove**, une application de rencontre où les utilisateurs créent un profil basé sur leurs goûts cinématographiques.

## 🚀 Lancer l'application

1. Assurez-vous que votre serveur PHP est actif.
2. Placez le dossier `app-loove` dans le répertoire racine de votre serveur (ex : `/var/www/html/`).
3. Accédez à l’application via votre navigateur :  
   👉 [http://localhost/app-loove/index.php?component=login](http://localhost/app-loove/index.php?component=login)

## 👤 Créer un compte

- Cliquez sur **Créer un compte**.
- Remplissez les informations demandées.
- Connectez-vous une fois l'inscription terminée.

## 🏠 Interface d’accueil (Home)

Une fois connecté :
- Créez votre **profil**.
- Choisissez **3 films** qui vous représentent.

Vous verrez alors la liste des autres profils disponibles à liker.

## 🧭 Navigation

### 🔼 En haut à gauche
- Accédez à **votre profil** pour le modifier ou vous déconnecter.

### 🔽 En bas (barre de navigation)
De gauche à droite :
1. 🎬 Modifier vos **films**
2. 💬 Accéder à vos **messages** (après un match)
3. 🔔 Voir vos **notifications**
4. ⚙️ Aller dans les **paramètres**

## ⚙️ Paramètres

Les paramètres vous permettent de :
- Rendre votre compte **privé**
- **Anonymiser** votre photo de profil
- Masquer votre **nom et prénom**
